# Video Walkthrough Script (5–8 min each project)

1. Hook (15s): Problem, stakes, target metric.
2. Methodology (30s): What {CRISP/SEMMA/KDD} enforces.
3. Data (60s): Source, schema, quality checks.
4. Notebook tour (2–3m): Key cells & results.
5. Deployment (1m): API/Gradio demo.
6. Lessons (60s): What worked & what to improve.
7. CTA (15s): Link to repo, Medium article.
